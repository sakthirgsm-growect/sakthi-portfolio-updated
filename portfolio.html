Voice AI Agent (Jarvis) – Telegram Voice & Text Assistant

Users often need quick answers, information retrieval, or task execution through messaging apps. However, traditional chatbots struggle to process both voice and text inputs, provide contextual responses, and deliver natural voice replies, creating a limited user experience.

Built an AI-powered Voice Assistant integrated with Telegram using n8n. The system intelligently processes both voice messages and text inputs, converts voice to text, generates intelligent AI responses, and optionally returns voice replies for a conversational assistant experience.

n8n · OpenAI GPT · OpenAI Whisper · Telegram Bot API · OpenAI TTS · Langchain nodes · SERP API

1 hour

Result: Built a multi-modal AI assistant capable of handling both voice and text interactions. Users can send voice messages via Telegram, which are automatically transcribed, processed by AI, and returned as intelligent responses. The system also generates natural AI voice responses, creating a conversational assistant experience similar to Jarvis-style voice agents.

n8n Gemini OpenAi Telegram Bot langchain Memory AI Agent SERP API
